getPublicKey

获取公钥#

检索由给定的 BIP32 路径派生的 BIP32 扩展公钥，向用户显示所请求密钥的描述，并要求其确认导出。

ES6

const result = await OneKeyConnect.getPublicKey(params);

CommonJS

OneKeyConnect.getPublicKey(params).then(function(result) {
});

参数#

可选的通用参数

导出单个公钥#

  • path - 必填 string | Array<number> 的最小长度为 1。 了解更多
  • coin - 可选 string 确定 coins.json 文件中指定的代币定义。可以使用币种的字段包含 coin_shortcut，coin_name 或 coin_label。如果未指定 coin，API将尝试从path获取网络定义。
  • crossChain - 可选 boolean 高级功能，仅当您知道自己在做什么时才使用它。 允许在链之间生成地址。例如，莱特币网络上的比特币路径将以莱特币格式显示跨链地址。

导出公钥包#

  • bundle - Array 对象的 path，coin 和 crossChain 字段

例子#

获取第五个 BTC 地址的公钥:

OneKeyConnect.getPublicKey({    path: "m/49'/0'/4'",    coin: "btc"});

获取多个 BTC 地址的公钥:

OneKeyConnect.getPublicKey({    bundle: [        { path: "m/49'/0'/0'" }, // account 1        { path: "m/49'/0'/1'" }, // account 2        { path: "m/49'/0'/2'" }  // account 3    ]});

返回值#

返回单个公钥

{    success: true,    payload: {        path: Array<number>, // hardended path        serializedPath: string, // serialized path        xpub: string,        // xpub in legacy format        xpubSegwit?: string, // optional for segwit accounts: xpub in segwit format        chainCode: string,   // BIP32 serialization format        childNum: number,    // BIP32 serialization format        publicKey: string,   // BIP32 serialization format        fingerprint: number, // BIP32 serialization format        depth: number,       // BIP32 serialization format    }}

了解有关 BIP32 的更多信息

返回多个公钥

{    success: true,    payload: [        { path, serializedPath, xpub, xpubSegwit?, chainCode, childNum, publicKey, fingerprint, depth }, // account 1        { path, serializedPath, xpub, xpubSegwit?, chainCode, childNum, publicKey, fingerprint, depth }, // account 2        { path, serializedPath, xpub, xpubSegwit?, chainCode, childNum, publicKey, fingerprint, depth }  // account 3    ]}

错误返回值

{    success: false,    payload: {        error: string // error message    }}
