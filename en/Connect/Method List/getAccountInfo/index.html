getAccountInfo

Get account info#

Gets an info of specified account.

ES6

const result = await OneKeyConnect.getAccountInfo(params);

CommonJS

OneKeyConnect.getAccountInfo(params).then(function(result) {
});

Params#

Optional common params

Using path#

  • path â€” required string | Array<number> minimum length is 3. read more
  • coin â€” required string determines network definition specified in coins.json file. Coin shortcut, name or label can be used.

Using public key#

  • descriptor â€” required string public key of account
  • coin â€” required string determines network definition specified in coins.json file. Coin shortcut, name or label can be used.

Using discovery#

BIP-0044 account discovery is performed and user is presented with a list of accounts. Result is returned after account selection.

  • coin â€” required string determines network definition specified in coins.json file. Coin shortcut, name or label can be used.

Other optional params#

params are forwarded to BlockBook backend using @onekeyhq/blockchain-link package

  • details â€” specifies level of details returned by request

    • basic (default) return only account balances, without any derived addresses or transaction history
    • tokens - response with derived addresses (Bitcoin-like accounts) and ERC20 tokens (Ethereum-like accounts), subject of tokens param
    • tokenBalances - same as tokens with balances, subject of tokens param
    • txs - tokenBalances + complete account transaction history

  • tokens â€” specifies which tokens (xpub addresses) are returned by the request (default nonzero)

    • nonzero - (Default) return only addresses with nonzero balance
    • used - return addresses with at least one transaction
    • derived - return all derived addresses

  • page â€” number transaction history page index, subject of details: txs

  • pageSize â€” number transaction history page size, subject of details: txs

  • from â€” number transaction history from block filter, subject of details: txs

  • to â€” number transaction history to block filter, subject of details: txs

  • gap â€” number address derivation gap size, subject of details: tokens

  • contractFilter â€” string Ethereum-like accounts only: get ERC20 token info and balance

  • marker â€” { ledger: number, seq: number } XRP accounts only, transaction history page marker

  • defaultAccountType â€” 'normal' | 'segwit' | 'legacy' Bitcoin-like accounts only: specify which account group is displayed as default in popup, subject of Using discovery

Example#

Get info about first bitcoin account

OneKeyConnect.getAccountInfo({    path: "m/49'/0'/0'",    coin: "btc",});

Get info about account using public key (device is not used)

OneKeyConnect.getAccountInfo({    xpub: "xpub6CVKsQYXc9awxgV1tWbG4foDvdcnieK2JkbpPEBKB5WwAPKBZ1mstLbKVB4ov7QzxzjaxNK6EfmNY5Jsk2cG26EVcEkycGW4tchT2dyUhrx",    coin: "btc",});

Get info about account using BIP-0044 account discovery

OneKeyConnect.getAccountInfo({    coin: "btc",});
